#### **🛡️ Security Containers**

**OpenSnitch Application Firewall**
```yaml
nexus-opensnitch:
  image: gustavo-iniguez-goya/opensnitch
  network_mode: host
  privileged: false
  user: "$(id -u):$(id -g)"

    - ~/.nexus-security/containers/opensnitch:/etc/opensnitchd:Z
```

```yaml
nexus-privoxy:
```
