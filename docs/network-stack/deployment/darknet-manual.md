## рҹ§… Core Anonymity Services

### рҹ§… Tor Network

**What is Tor?** рҹӨ”
Tor (The Onion Router) routes your traffic through multiple relays, encrypting at each hop. Nobody can trace your connection!

**Configuration:** `configs/torrc`

```bash
# рҹҡҖ Start Tor
./nexus-darknet start tor

# рҹӣ‘ Stop Tor
./nexus-darknet stop tor

# рҹ“Ҡ Check Tor status
curl --socks5 127.0.0.1:9050 https://check.torproject.org/api/ip
```

**Ports:**
| Port | Protocol | Purpose |
|------|----------|---------|
| 9050 | SOCKS5 | Main proxy port |
| 9051 | TCP | Control port |

**Usage Examples:**
```bash
# рҹҢҗ Browse anonymously
curl --socks5 127.0.0.1:9050 https://example.com

# рҹ§… Access .onion sites
curl --socks5 127.0.0.1:9050 http://duckduckgogg42xjoc72x3sjasowoarfbgcmvfimaftt6twagswzczad.onion

# рҹ”§ Configure Firefox
# Network Settings вҶ’ Manual Proxy вҶ’ SOCKS Host: 127.0.0.1:9050
```

### рҹ”— I2P Network

**What is I2P?** рҹӨ”
I2P (Invisible Internet Project) uses "garlic routing" - bundling multiple messages together for extra anonymity. Perfect for hidden services!

**Configuration:** `configs/i2pd.conf`

```bash
# рҹҡҖ Start I2P
./nexus-darknet start i2p

# рҹҢҗ Access I2P console
# http://127.0.0.1:7070
```

**Ports:**
| Port | Protocol | Purpose |
|------|----------|---------|
| 4444 | HTTP | HTTP proxy for .i2p sites |
| 4447 | SOCKS | SOCKS proxy |
| 7070 | HTTP | Web console |
| 7656 | TCP | SAM API for applications |

**Usage Examples:**
```bash
# рҹ”— Browse I2P eepsites
curl --proxy 127.0.0.1:4444 http://i2p-projekt.i2p

# рҹ”— Access I2P forum
curl --proxy 127.0.0.1:4444 http://zzz.i2p
```

**вҡ пёҸ Important:** I2P takes 5-10 minutes to build tunnels on first start!

### рҹ•өпёҸ Privoxy - Smart Router

**What is Privoxy?** рҹӨ”
Privoxy is your intelligent traffic router! It automatically sends:
- `.onion` domains вҶ’ Tor
- `.i2p` domains вҶ’ I2P
- Everything else вҶ’ Tor (for anonymity)

**Configuration:** `configs/privoxy.conf`

```bash
# рҹҡҖ Start Privoxy
./nexus-darknet start privoxy
```

**Port:** `8118` (HTTP)

**Usage - ONE PROXY FOR EVERYTHING:**
```bash
# рҹҢҗ Regular sites (goes through Tor)
curl --proxy 127.0.0.1:8118 https://example.com

# рҹ§… Onion sites (automatically routed to Tor)
curl --proxy 127.0.0.1:8118 http://example.onion

# рҹ”— I2P sites (automatically routed to I2P)
curl --proxy 127.0.0.1:8118 http://example.i2p

# вңЁ Set as system proxy and forget about it!
export http_proxy=http://127.0.0.1:8118
export https_proxy=http://127.0.0.1:8118
```

**рҹҺү Privoxy is the recommended proxy** - use it for everything!

## рҹҢІ Mesh Networks

### рҹҢІ Yggdrasil

**What is Yggdrasil?** рҹӨ”
Yggdrasil creates an encrypted IPv6 mesh network. Every node gets a unique IPv6 address derived from its public key. Works over the internet or local networks!

**Configuration:** `configs/yggdrasil.conf`

```bash
# рҹҡҖ Start Yggdrasil
./nexus-darknet start yggdrasil

# рҹ“Ҡ Check your Yggdrasil address
curl http://127.0.0.1:9002/getself
```

**Ports:**
| Port | Protocol | Purpose |
|------|----------|---------|
| 9001 | TCP/TLS | Peer connections |
| 9002 | TCP | Admin API |

**Features:**
- рҹ”җ End-to-end encrypted
- рҹҢҗ Works over internet OR local mesh
- рҹ“Қ Automatic peer discovery
- рҹ”„ Self-healing routing

**What is Reticulum?** рҹӨ”
Reticulum is a cryptographic mesh networking stack designed for resilient communication. Works over ANY transport - radio, serial, TCP, I2P!

**Configuration:** `configs/reticulum.conf`

**Interfaces Configured:**

- рҹ”— `I2PInterface` - Anonymous mesh (optional)

### рҹҰҮ BATMAN-adv

**What is BATMAN-adv?** рҹӨ”
BATMAN-adv (Better Approach To Mobile Adhoc Networking) creates a Layer 2 mesh network. Your devices appear to be on the same local network even across WiFi hops!

```bash
# рҹҰҮ Enable BATMAN-adv mesh

# рҹ“Ҡ Check mesh status
```

**Requirements:**
- рҹ”§ `batman-adv` kernel module
- рҹ“¶ WiFi interface in ad-hoc or mesh mode

**Mesh Interface:** `bat0` with IP `192.168.199.1/24`

## рҹҢҗ Distributed Storage

### рҹҢҗ IPFS

**What is IPFS?** рҹӨ”
IPFS (InterPlanetary File System) is a distributed file system. Content is addressed by its hash - the same file has the same address everywhere!

```bash
# рҹҡҖ Start IPFS
./nexus-darknet start ipfs

# рҹ“Ө Add a file
ipfs add myfile.txt

# рҹ“Ҙ Get a file by hash
ipfs get QmXYZ123...

# рҹҢҗ Access web gateway
```

```bash
./nexus-darknet start tor
```

### вқҢ Problem: Tor Not Connecting

```bash
# 1пёҸвғЈ Check Tor logs
podman logs nexus-tor 2>&1 | tail -50

# 2пёҸвғЈ Common issues:
#    - Clock not synchronized вҶ’ Run: sudo ntpd -gq

#    - ISP blocking Tor вҶ’ Use bridges (see Snowflake)
```

**This is normal!** I2P needs 5-10 minutes on first start.

```bash
# Check I2P console for progress
# http://127.0.0.1:7070

curl -s http://127.0.0.1:7070 | grep -i tunnel
```

```bash
# 1пёҸвғЈ Check if Tor is running (Privoxy depends on it)

# 2пёҸвғЈ Check Privoxy config syntax
podman exec nexus-privoxy privoxy --config-test /etc/privoxy/config

# 3пёҸвғЈ Check Privoxy logs
podman logs nexus-privoxy
```

```bash
# 1пёҸвғЈ Check if Flask is installed
```

## вқ“ FAQ

### рҹӨ” General Questions

**Q: Is this legal?**
> вң… Yes! Using Tor, I2P, and mesh networks is legal in most countries. These tools protect privacy and enable free speech.

**Q: Will this slow down my internet?**
> вҡ пёҸ Anonymity networks add latency. Tor adds ~200-500ms. Use HAProxy load balancing for better speeds.

**Q: Can my ISP see I'm using Tor?**
> рҹ”Қ They can see you're connecting to Tor, but not what you're doing. Use bridges or Snowflake to hide Tor usage.

### рҹ”§ Technical Questions

**Q: How much disk space do I need?**
> рҹ’ҫ The stack itself needs ~500MB. Container images add ~2GB. IPFS storage varies by usage.

**Q: Can I run this on a Raspberry Pi?**
> рҹҘ§ Yes! Works on any Linux. RPi 4 (4GB+) recommended for full stack.

**Q: How do I add more Tor circuits?**
> рҹҗҷ Use the Hydra system: `./hydra/start-tor-hydra.sh` creates 9 circuits!

**Q: Why is I2P so slow?**
> рҹ”— I2P prioritizes anonymity over speed. It gets faster after running for a while as it learns the network.

### рҹӣЎпёҸ Security Questions

**Q: Is this 100% anonymous?**
> вҡ пёҸ No system is perfect. Operational security matters. Don't log into personal accounts, don't download files carelessly.

**Q: Should I use a VPN with Tor?**
> рҹӨ” Generally no. VPN+Tor can actually reduce anonymity. Tor alone is designed for anonymity.

**Q: How do I know if it's working?**
> вң… Run `./nexus-darknet health` or visit https://check.torproject.org through the proxy.
