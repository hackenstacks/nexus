# Contributing to NeXuS Network Stack

**"Together Everyone Achieves More"** ⚡⚡⚡

## How to Contribute

### Keep It Simple
If grandma can't use it, it's too complex!

### Test Your Changes
```bash
./nexus-doctor.sh      # Check everything works
./nexus-status.py      # Test the monitor
./nexus-launch.sh      # Make sure it launches
```

### License
All contributions are GPL-3.0:
- Keep it open source

### Submit
1. Fork the repo
2. Make your changes
3. Test thoroughly
4. Submit a Pull Request
