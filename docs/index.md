## What Is NeXuS

NeXuS is alternative economic infrastructure for individuals. As AI displaces workers, NeXuS routes value back — your hardware, your content, your data, your participation generates income that flows directly to you. No intermediary capturing the margin.

**NeXuS value is the network. Users make up the network. Therefore users ARE the value.**

REVERSED.

Users create the system.

## The Stack

| Layer | What it does |
|-------|-------------|
| Transport | Tor + I2P + Reticulum + Yggdrasil — anonymous, censorship-resistant |
| Privacy | Privoxy + HAProxy + DNS-over-Tor — zero leaks by default |
| Node Services | NeXuS chain + IPFS + DIVA connector + contribution monitor |
| Interface | NeXuS Wallet — the Command Center, browser into the network |
| Social | Townhall marketplace + NFT Forge + badge / reputation system |
