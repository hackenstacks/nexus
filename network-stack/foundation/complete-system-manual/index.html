    • Democratic Security: Tools that work from "Granny to Power User"
    • CLI-First Philosophy: Terminal-native with fire aesthetics
    • AI Collaboration: Multi-AI harmony and consciousness bridging
    • Backup-First Protocol: Fearless innovation through safety nets

    🛡️ Security Containers¶

    OpenSnitch Application Firewall 

    nexus-opensnitch:
  image: gustavo-iniguez-goya/opensnitch
  network_mode: host
  privileged: false
  user: "$(id -u):$(id -g)"
  volumes:
    - ~/.nexus-security/containers/opensnitch:/etc/opensnitchd:Z

    Privoxy + uBlock Origin 

    nexus-privoxy:
  image: privoxy/privoxy
  ports: ["8118:8118"]
  user: "$(id -u):$(id -g)"
  volumes:
    - ~/.nexus-security/containers/privoxy:/etc/privoxy:Z
  # Features:
  #   ✅ uBlock Origin rule integration
  #   ✅ Privacy header manipulation
  #   ✅ Tracking protection

    Tor Stealth Network 

    nexus-tor:
  image: alpine/tor:latest
  ports: ["9050:9050", "9051:9051"]
  user: "$(id -u):$(id -g)"
  volumes:
    - ~/.nexus-security/containers/tor:/etc/tor:Z
  # Features:
  #   ✅ Multi-circuit SOCKS proxy
  #   ✅ Exit node selection
  #   ✅ Performance optimization
